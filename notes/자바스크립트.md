## 변수 
- 변수는 프로그래밍 언어의 가장 기본적인 개념
- 변하는 값 의미
- 특정 데이터를 담는 용기나 그릇과 같음
- 자바스크립트에서 변수는
	- var a;
	- 변수를 선언하고 아무 값도 할당하지 않으면 기본적으로 undefined라는 초기 값을 가짐
- 변수를 선언하고 나면 변수 안에 원하는 값 정의 가능
	- var a;
	- a = 10;
	- 위 코드는 a라는 변수를 정의하고 이 변수에 10이라는 숫자를 할당한 코드
	- 자바스크립트는 변수의 타입을 정의하지 않아도 코드를 실행할 때 자동으로 타입 결정
- 변수는 영어와 한글 모두 사용 가능
	- 숫자로 시작되어서는 안됨
	- 대소문자가 구분됨
	- 변수명에 예약어를 사용할 수 없음
- 변수의 타입은 typeof 라는 예약어로 확인 가능
	- typeof 변수;

## 문자열
- 문자열은 변수의 여러 타입중 하나
- 문자열은 작은 따옴표 혹은 큰 따옴표로 정의 가능
- 변수.length라는 예약어는 자바스크립트에서 변수의 타입에 따라 문자열, 배열의 길이를 수자 형태로 확인 할 수 있음

## 조건문
- 조건문은 주어진 조건을 만족할 때만 코드를 실행하게 하는 문법
- 조건에 따라 다른 동작을 실행하고 싶을 때 사용
- if 조건문은 주어진 조건식이 참이면 if 문 블록 안에 있는 코드를 실행
	- if ( 조건식 ) { 코드 };
- if else 문은 주어진 조건식이 참이 아니면 else 문 블록 안의 코드가 실행됨
- else if 문은 조건식이 여러 개인 경우 사용할 수 있는 문법 

- switch문은 조건식이 여러 개인 경우에 사용할 수 있는 조건문
	- switch ( 변수 ) { case 값 1: 변수가 값 1이면 실행할 로직; break;
	 case 값 2: 변수가 값 2이면 실행할 로직; break;
	 default: 변수가 위 어떤 케이스에도 해당하지 않는 경우 실행할 로직; }

## Object 객체
- 자바스크립트는 객체 기반 언어
- 객체는 key와 value의 형태
- var obj = { 객체의 내용 };
	- 위 코드는 obj라는 변수에 객체를 새로 할당한 코드
	- { }라는 기호 자체가 객체를 의미하며 이를 객체 리터럴이라고 함
	- 일반적으로 객체를 생성할 때는 객체 리터럴을 사용하여 선언
- 객체를 선언하고 나면 속성을 추가할 수 있음
	- var obj = { };
	- obj.num = 10; 또는 obj[ 'num' ] = 20;
- 이미 정의한 속성 값을 변경하는 것도 가능
	- var obj = { };
	- obj.num = 10;
	- obj.num = 20;

## Map
- Map 객체는 키와 값을 한 쌍으로 이루어진 컬렉션
- Map 객체는 Map 생성자 함수로 생성
- const map = new Map([ ['key', 'value'], ['key', 'value'] ]);
- Map 객체는 데이터를 추가할 때는 Map.prototype.set 메서드를 사용
	- const map = new Map();
	- map.set ('key1', 'value1'); 
	- map.set ('key2', 'value2');
- Map 값을 삭제 하려면  Map.prototype.delete 메서드를 사용
	- map.delete( 'key1' );
- 모든 값을 삭제할 때는 Map.prototype.clear 메서드를 사용
	- map.clear( );
- map.get(key): key에 해당하는 값을 반환
- map.has(key): key가 존재하면 true, 존재하지 않으면 false를 반환
- map.size: 요소의 개수를 나타냄

## set
- set 객체는 중복되지 않는 값들의 집합
- set을 통해 교집합, 합집합, 차집합 등 수학적 집합 표현 가능
- const set = new Set( );
- Set객체에 값을 추가할 때는 Set.prototype.add메서드를 사용
	- set.add('set 값');
	- set은 중복된 값의 추가를 허용하지 않음
- Set 값을 삭제 하려면  Set.prototype.delete 메서드를 사용
	- set.delete( 'set 값' );
- 모든 값을 삭제할 때는 Map.prototype.clear 메서드를 사용
	- set.clear( );
- set.has(value): value가 존재하면 true, 존재하지 않으면 false를 반환
- set.size: Set 객체가 가진 원소의 수를 나타냄

## Array 배열
- 배열은 객체와 더불어 실제로 웹 애플리케이션을 구현할 때 가장 많이 쓰이는 변수 타입
- var arr = [];
- 객체는 속성/값의 조합으로 데이터를 저장하지만 배열은 인덱스/값의 조합으로 데이터를 저장
	- var arr = [];
	- arr[0] = 10;
	- 배열의 인덱스는 0부터 시작, 빈 배열에 최초로 값을 추가하면 0번째 인덱스에 값이 추가됨
- 자주 사용하는 배열 API
	-  arr.push( ): 배열에 데이터 추가 (맨 끝 인덱스부터 추가됨)
	- arr.slice(): 배열의 특정 인덱스에 있는 값을 반환 (배열의 내용이 변환되지 않음)
	- arr.splice(0, 1, 10): 배열의 특정 인덱스에 있는 값을 변경 또는 삭제 (배열의 내용이 변경됨)
	- arr.pop( ): 배열의 마지막 인덱스의 값을 꺼냄 (배열의 내용이 변경됨)
	- shift( ): 배열의 첫번째 인덱스의 값을 꺼냄 (배열의 내용이 변경됨)

## Loop 반복문
- 반복문은 특정 로직을 반복할 때 사용하는 문법
- for (반복할 변수 초기화; 반복조건; 반복할 변수의 증감식) { 반복할 연산 }
	- for (var i = 0; i < 10; i++) { console.log(i); // 0부터 9까지 출력 }
- forEach( ): 배열에 사용하기 좋은 반복문
	- var arr = [10, 20, 30];
	- arr.forEach(function(value, index) {console.log('array index: ' + index + ' value : ' + value);});
- for in 반복문: 배열과 객체에 모두 사용할 수 있는 반복문
	- 배열의 인덱스를 1개씩 접근하여 순차적으로 순회하는 경우
		for (var key in arr) {
	  console.log(arr[key]);}
	- 객체의 키를 1개씩 접근하여 순차적으로 순회하는 경우
		for (var key in obj) {
	  console.log(obj[key]);}
- for of 반복문: 반복 가능한 속성을 가지는 컬렉션에 사용하기 좋은 반복문
	반복 가능한 속성은 객체가 [Symbol.iterator] 메서드를 가지고 있어야 하고, [Symbol.iterator] 메서드는 interior 객체를 반환해야 함
## for in vs for of
- var arr = [10, 20, 30 ]
- 배열을 각각 순회하는 경우
	for (var num of arr) {
    console.log(num); }
     -> 배열의 값 자체에 접근
     for of는 반복 가능한 배열의 요소를 순회하기 때문에 data를 직접 순회


	for (var num in arr) {
    console.log(num); }
	 -> 배열의 인덱스에 접근
	 for in 은 객체의 속성을 순회하기 때문에 배열의 속성이자 key에 해당하는 index를 반환하며 순회

## 연산자
- 연산자는 프로그래밍 로직을 구현할 때 논리식이나 산술식을 표현하기 위해 사용하는 기호
- 산술 연산자: 숫자를 계산하는 연산자
	- +, -, *, /, %
- 논리 연산자: 참 거짓을 다루는 연산자
	- || : or, 또는
	- &&: 그리고 
- 조건 연산자: 조건에 따라 값을 하나 고르는 연산자. if/ else를 하나로 줄여 쓴 표현식
	- ? :
	- 조건 ? 참일 때 값 : 거짓일 때 값
	- 조건이 참이면 앞의 값을 사용하고 거짓이면 뒤의 값 사용
	- let age = 20;
	- let result = age >= 19 ? "성인" : "미성년자";

- 관계 연산자
	- >, <
	- === : 일치 연산자 

## 함수
- 함수는 특정 기능을 수행하는 코드의 단위
- 즉 여러 줄이 모여 있는 코드 모음 정도
- 함수 표현식
	- var sumNumbers = function(a, b) { return a + b; };
- 함수 선언문
	- function sumNumbers(a, b)  { return a + b; }
- 함수를 작성할 때는 가급적 단일 책임 원칙을 지키는 것이 좋음
	- 1개의 함수는 1개의 기능만 담당해야 한다는 원칙
	- 하나의 로직 외에 다른 로직들이 뒤엉켜 있으면 코드를 재사용하기 쉽지 않음

## This
- this는 함수의 실행 컨텍스트를 가리키는 예약어
- 함수가 실행되는 환경, 함수가 실행될 때의 컨텍스트
- 첫 번째 this
	- 글로벌(전역) 컨텍스트
	- 일반 함수의 this는 전역 컨텍스트
	- console.log(this); // window
	- 여기서 출력된 window는 자바스크립트의 최상위 객체를 가리킴
- 두 번째 this
	- var obj = {
		num: 10,
		printNum: function() {
		console.log(this.num); } };
	- obj.printNum(); // 10
	- 객체 속성 함수 안에서의 this는 기본적으로 해당 객체를 가리킴
- 세 번째 this
	- 생성자 함수에서의 this는, new로 인스턴스를 생성하는 순간 함수가 실행되기 때문에 함수 내부를 가리킴
- 네 번째 this
	- 데이터를 받아올 때 사용하는 HTTP 요청과 같은 비동기 처리 코드
	- 실제로 웹 개발을 할 때 가장 많이 마주하게 되는 this
	- 기본적으로 HTTP 요청과 같은 비동기 처리 코드는 전역 컨텍스트를 가짐
	- 비동기 처리는 요청을 보내고 기다리지 않음
		- 다음 코드 바로 실행
		- 결과는 나중에 도착
		- fetch는 대표적인 비동기 코드
	- 비동기 처리 코드의 콜백함수는 전역 컨텍스트를 가리킴

## 프로토타입
- 자바스크립트는 프로토타입 기반 언어
- 어떤 객체를 원형으로 삼고 이를 참조함으로써 상속과 비슷한 효과를 얻음
- 인스턴스를 생성하면  `__proto__` 라는 속성이 자동으로 부여되는데 그 속성은 생성자 함수 array의 prototype 속성을 참조
- array.prototype
	- prototype의 타입은 객체이고, prototype을 참조하는 `__proto__` 또한 객체
	- prototype 내부에는 인스턴스가 사용할 메서드와 속성을 저장
	- 인스턴스에서 자동으로 생성한 `__proto__` 속성을 통해 prototype 내부에 정의된 메서드와 속성을 사용할 수 있음
- 자바스크립트는 함수에 자동으로 prototype이라는 객체 타입의 속성을 생성
- new연산자와 함께 함수를 호출하여 함수를 생성자 함수로써 사용할 경우, 생성된 인스턴스에는 숨겨진 속성인 `__proto__` 속성이 자동으로 생성
- `__proto__` 속성은 생성자 함수의 prototype 속성을 참조
- `__proto__` 속성은 생략이 가능하기 때문에 인스턴스에서 생성자 함수의 prototype에 정의된 메서드나 속성에 접근할 수 있음

## 스코프 Scope
- 변수의 유효 범위
- 글로벌 스코프
	- 다른 프로그래밍 언어와는 다르게 자바스크립트의 변수는 유효 범위가 전역으로 시작
	- 변수를 선언한 후 함수를 만들어 함수 내에서 그 변수에 접근하더라도 같은 값
- 로컬 스코프
	- 함수 안에서 변수를 새로 선언하는 경우 함수 단위의 지역 범위인 함수 스코프를 가짐
	- 변수의 유효 범위는 함수 단위로 한정됨
- 스코프 체인
	- 변수를 찾을 때 먼저 자신이 속한 스코프에서 찾고 없으면 상위 스코프에서 찾는 현상을 스코프 체인이라고 함
	- 함수가 실행되면 먼저 지역 범위인 함수 안에서 변수를 찾고 함수 안에 변수가 없으면 상위 스코프인 전역 범위에서 변수를 다시 찾음
- 렉시컬 스코프
	- 자바스크립트는 함수를 어디서 선언하였는지에 따라서 상위스코프를 결정하는 렉시컬 스코프 규칙을 따름


## 클로져
- 클로져는 함수의 실행이 끝난 뒤에도 함수에 선언된 변수의 값을 접근할 수 있는 성질
- 다른 언어와 차별되는 자바스크립트의 특징
- 함수를 변수나 인자로 넘길 수 있는 자바스크립트의 성질(일급객체)
- private 변수
	- 일반적으로 프로그래밍에서 외부에서 사용하지 않거나 접근하면 안 되는 변수와 함수는 private로 선언하여 사용함
