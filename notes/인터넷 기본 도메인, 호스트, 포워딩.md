
## 도메인
- ip: 인터넷에 연결되어 있는 장치들이 가진 각각의 장치를 식별할 수 있는 주소
- ip는 사람이 이해하고 기억하기 어렵기 때문에 이를 위해 부여된 이름이 도메인
- 호스트 설정: 도메인에 연결되어있는 ip 등록

## 포워딩
- 도메인으로 접근한 사용자를 다른 도메인으로 보내는 행위
- 유동 포워딩/ 고정 포워딩

## host 파일
브라우저가 도메인에 해당하는 ip를 찾는 순서
1.  local cache 검색
2. host 파일 검색
3. 도메인 네임서버 검색

## 포트포워딩
- 클라이언트와 서버가 소통하기 위한 여러 입구 
- 웹서버 식별하는 포트
- 외부에서 특정 IP 주소와 포트로 들어오는 네트워크 패킷을 내부의 다른 IP 주소와 포트로 전달하는 기술
- 포트포워딩 설정을 통해 외부에서 내부 서버에 접속하거나, 특정 서비스에 접근
- DDNS는 동적으로 변하는 외부 IP 주소를 고정된 도메인 주소로 연결해주고, 포트 포워딩은 외부에서 들어오는 특정 포트 요청을 공유기 내부의 특정 기기의 포트로 연결
